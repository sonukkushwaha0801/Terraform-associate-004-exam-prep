# Maintain Infrastructure with Terraform

## Domain Overview

After infrastructure is deployed, Terraform provides tools to:

- Import Existing Infrastructure
- Inspect Terraform State
- Troubleshoot Terraform Operations

This domain focuses on maintaining infrastructure that already exists rather than creating new infrastructure.

Although smaller than previous domains, HashiCorp frequently tests:

- terraform import
- terraform state commands
- TF_LOG
- Debugging Workflows
- State Inspection

# Why Infrastructure Maintenance Matters

Terraform is not only a provisioning tool.

Real-world teams spend significant time:

- Troubleshooting
- Auditing
- Refactoring
- Importing Legacy Resources
- Investigating Failures

Terraform provides CLI tools for these operational tasks.
